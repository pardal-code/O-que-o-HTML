Como Surgiu

(Voltar)

O HTML (Hyper Text Markup Language), ou em nosso bom e velho português, (Linguagem de Marcação de Hipertexto), é uma linguagem que foi introduzida nos primórdios da internet por um cientista chamado Tim Berners-Lee em meados de 1991. Foi nesse período que se originou a primeira versão do HTML, e em 6 de agosto daquele mesmo ano, foi publicada a primeira página web, na qual Tim explicava como funcionava a nova ferramenta.

Mal sabia ele que aquilo seria um pontapé inicial para a internet como a conhecemos hoje, pois todos os sites que você visita são construídos usando HTML. Esta linguagem de marcação é a espinha dorsal da web, permitindo a criação de páginas estruturadas com links que conectam vastas quantidades de informações.

Com o passar dos anos, o HTML evoluiu significativamente. A World Wide Web Consortium (W3C), fundada por Berners-Lee, assumiu a responsabilidade de desenvolver padrões para a web, incluindo o HTML. Desde então, várias versões foram lançadas, adicionando novos elementos e atributos para enriquecer a experiência do usuário na web, culminando no HTML5, que é a versão mais recente e robusta, suportando vídeo, áudio, gráficos e animações interativas.

A evolução do HTML reflete o crescimento dinâmico da internet e continua a ser um campo ativo de desenvolvimento, prometendo novas funcionalidades e melhorias que moldarão o futuro da navegação na web.

Afinal Para Que Serve O HTML

(Voltar)

O HTML é a linguagem básica para criar sites. Ele funciona como o esqueleto de uma página na internet, organizando onde as coisas vão aparecer, como textos, imagens e botões.

Graças ao HTML, podemos ter sites com páginas que se ligam umas às outras através de links, criando essa grande rede que é a internet. É fácil de aprender e é o primeiro passo para quem quer começar a criar seus próprios sites.

Não captou a ideia? Imagine que o HTML é a fundação de uma casa. Sem ele, não teríamos onde colocar as paredes, janelas ou portas. No mundo dos sites, o HTML define onde cada parte do conteúdo deve ficar, como um título, um parágrafo ou uma imagem. Ele também cria os caminhos que conectam diferentes páginas da web, como se fossem os corredores que ligam os cômodos da casa. É a base que permite que a internet seja um lugar cheio de informações e recursos que podemos explorar com apenas alguns cliques.

As Principais Tags

(Voltar)

As tags HTML são os elementos básicos da linguagem de marcação e servem para definir a estrutura e o conteúdo de uma página web. As tags são sempre abertas com os sinais de "<Insira a tag aqui>" e fechadas com "<Insira a tag aqui/>". Aqui estão algumas das principais tags HTML e suas funções:

Tags de Estrutura e Cabeça

  • "<DOCTYPE html>": Define o tipo de documento e a versão do HTML.
  • "<html>": É a raiz do documento HTML e contém todo o conteúdo da página.
  • "<head>": Contém metadados e links para scripts e folhas de estilo.
  • "<title>": Define o título da página, que aparece na aba do navegador.

Tags de Corpo

  • "<body>": Engloba todo o conteúdo visível da página, como texto, imagens e links.
  • "<h1>" a <h6>": São usadas para títulos e subtítulos, com "<h1>" sendo o mais importante.
  • "<p>": Define um parágrafo de texto.
  • "<a>": Cria um hiperlink para outra página ou recurso.
  • "<img>": Insere uma imagem na página.
  • "<ul>", "<ol>", "<li>": Criam listas não ordenadas e ordenadas, com itens de lista.
